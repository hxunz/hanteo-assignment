# hanteo-assignment

### 실행 방법
```
npm install
npm run dev
```

- 처음 홈 페이지 진입 후 상단에 카테고리 버튼을 통해서 각 카테고리에 맞는 페이지를 확인할 수 있습니다.
- 차트 카테고리의 배너를 클릭하면 각 배너에 맞는 외부 링크로 이동하게 됩니다.
- 좌우 슬라이드를 통한 카테고리 이동은 회색으로 되어있는 컨텐츠 영역에서 실행 할 수 있습니다.

### 추가 기능 구현
- 현재 페이지의 카테고리 버튼 클릭 시 현재 페이지의 최상단으로 이동하도록 하였습니다.
- footer를 추가하였습니다.
